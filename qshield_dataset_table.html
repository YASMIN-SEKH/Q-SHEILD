CMU Keystroke Benchmark Killourhy & Maxion, DSN 2009 · CMU KeystrokePrimary 51 400 per user (8 sessions × 50) Hold time (H) Down-Down (DD) Up-Down (UD) 31 features Gold-standard fixed-text benchmark. Password ".tie5Roanl". Direct input to Q-Shield FFT encoder. Best for establishing baseline EER. Free — no sign-up cs.cmu.edu/~keystroke Kaggle mirror
Aalto Mobile Keystroke DB Dhakal et al. · Aalto University Keystroke 37,370 15 sentences per user Hold latency (HL) Inter-key latency (IL) Press latency (PL) Release latency (RL) Largest public keystroke DB. Mobile free-text transcription. Used by TypeFormer (EER 3.25%). Tests Q-Shield encoder on mobile scenario. Free — direct download userinterfaces.aalto.fi
Balabit Mouse Dynamics Fülöp et al., 2016 · Balabit / One Identity MousePrimary 10 Train + test splits X, Y coordinates Timestamp Button state Move / Click / Drag Standard mouse dynamics benchmark. Remote desktop sessions. Git clone — no account needed. Used in nearly all mouse biometrics papers. Free — git clone github.com/balabit/Mouse-Dynamics-Challenge
DFL Mouse Dynamics Antal & Denes-Fazakas · Sapientia University Mouse ~30 Multiple sessions Timestamp X, Y position Button events Same format as Balabit Structurally identical to Balabit. Completely free, no restrictions. Used for cross-dataset validation and transfer learning experiments. Free — no restrictions ms.sapientia.ro/~manyi/mousedynamics
HMOG Dataset Sitová et al., IEEE TIFS 2016 · William & Mary TouchPrimary 100 24 per user (8 × 3 tasks) Touch x, y, pressure, area Accelerometer (3-axis) Gyroscope (3-axis) Swipe + tap + keystroke Gold-standard mobile touch benchmark. Reading, writing, navigation tasks. Sitting + walking conditions. EER 7.16% (walking). 2–6 hrs/subject. Free — ToU agreement hmog-dataset.github.io/hmog github.com/hmog-dataset/hmog
HuMIdb Acien et al., 2021 · BiDAlab, UAM Touch 600 8 tasks per user Swipe + tap Accelerometer Gyroscope Magnetometer Fixed + free-text KS Handwriting (digits) Largest + most diverse mobile DB. 14 sensors, 300+ device models, 14 countries, unsupervised. Best for cross-device generalization of Q-Shield encoder. Email request github.com/BiDAlab/HuMIdb Email signed license to atvs@uam.es
BB-MAS Dataset Belman et al., 2019 · Syracuse / FIU Multimodal 117 Desktop + Tablet + Phone Fixed + free-text keystroke Touch swipe Gait (walk, stairs) Accelerometer + Gyroscope 3 devices per subject Unique: typing, swipe, gait from same 117 subjects across 3 devices. 3.5M keystroke events + 57.1M IMU data points. Best for cross-modality fusion validation. Author contact arxiv.org/abs/1912.02736 Contact corresponding author via paper
UMDAA-02 Mahbub et al., BTAS 2016 · Univ. of Maryland Multimodal 48 ~248 sessions/user over 2 months Touchscreen Gyroscope + Accel. Magnetometer Front camera GPS + WiFi + Bluetooth Long-duration continuous dataset (2 months, same device). 14 sensors. Best for testing temporal drift and session-long behavioral consistency in Q-Shield. Author request arxiv.org/abs/1610.07468 Request via UMD GAMMA lab
